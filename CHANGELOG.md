## Changelog for 1.1.6

### **Slot Locking & Binding**
**New**
- Added **Slot Binding**: link an inventory slot to a hotbar slot, then shift-click it to instantly swap the item onto that hotbar slot — handy for swapping tools or gear in one click. Hold the Slot Edit key (default **L**) and left-click an inventory slot, then a hotbar slot, to create the link. Each link shows a coloured line, with the inventory slot and its target hotbar slot outlined in different colours.
- Remove a link: while holding the Slot Edit key, left-click the inventory slot to remove the link.

### **Mining Enhancements**
**Fixes**
- The Ping Offset Mining overlay now turns a steady green at the moment to switch blocks, instead of flickering and rarely showing the green at all.
- The overlay colour pickers and highlight opacity slider now actually change how the overlay looks — before, your colour choices were ignored.
- The overlay no longer briefly blinks off at the instant you start mining a block you were looking at.
