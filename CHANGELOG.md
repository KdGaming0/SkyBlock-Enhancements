New Features:

- Item Outline Glow on dropped items, matching their rarity
- RemindMe System: Custom reminder command with multiple trigger types
  - Create one-time or repeating reminders with `/remindme create`
  - Support for real-time and while-playing (in-game time, pause when you leave) triggers
  - Multiple output options: chat, title box (on screen), or both
  - List active reminders with `/remindme list`
  - Remove reminders individually or all at once
  - Built-in help documentation with `/remindme help`
