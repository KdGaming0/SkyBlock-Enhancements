## Changelog for 1.1.6

### **Slot Locking & Binding**
**Changes**
- You can now also create a link by holding the Slot Edit key and **dragging** an item straight onto the hotbar slot you want it bound to — whichever feels more natural.
- Shift-clicking the **hotbar slot** now works too: the linked item drops straight into it, even when that hotbar slot is empty (previously only the inventory slot side responded).
